Shortcutting for Driftwood.

## Requirements

- iOS 8.0+/macOS 10.11+/tvOS 9.0+
- Swift 5.0

## Pods

```ruby
pod 'DriftwoodHandily'
```

## Usage

Same as [Driftwood](https://github.com/wlgemini/Driftwood), but remove all `dw` prefix for it.
